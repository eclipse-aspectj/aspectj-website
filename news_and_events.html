April 7, 2005 AspectJ 5 M2 build now available
The second milestone build of AspectJ 5 is now available, with full support for compiling Java 5 language features and annotation matching and binding. See the README for more details, or go straight to the AspectJ 5 Developer's Notebook for a detailed description of the new language features.

January 19th, 2005 - AspectJ and AspectWerkz to join forces.
We're excited to announce that the Aspectj and Aspectwerkz teams have agreed to join forces and work together on the AspectJ 5 release. Read more in "AspectJ and AspectWerkz to join forces".
January 18th, 2005 - Two new AspectJ books now shipping.
The books Eclipse AspectJ: Aspect-oriented programming with Eclipse and the AspectJ Development Tools by Adrian Colyer, Andy Clement, George Harley and Matthew Webster, and The AspectJ Cookbook by Russ Miles are now shipping.
January 17th, 2005 - AspectJ 5 development builds now support full Java 5 source compilation
The AspectJ 5 development builds as of 17th January now include full support for source compilation of Java 5 programs.
December 10th, 2004 - AspectJ 5 M1 (milestone 1) build now available.
The first milestone build of AspectJ 5 is now available, with support for binary weaving of Java 5 classes. This build supports annotations, enums, varargs, autoboxing, covariance, and detection of bridge methods. See the README for more details, or go straight to the AspectJ 5 Developer's Notebook for a detailed description of the new language features.
November 9th, 2004 - "Easy Install" AspectJ and AJDT distributions available.
Russ Miles has packaged together integrated installers for Eclipse, AJDT, and AspectJ on the Windows and Mac platforms. So now you can install a fully-configured AspectJ development environment in one single step. Also look out for his forthcoming book, the AspectJ Cookbook.
