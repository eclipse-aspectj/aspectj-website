AspectJ 1.8.14 Readme

Small release that:

  • Will skip module-info.class and classfiles under META-INF that you find in multi release jars (as described in https://bugs.eclipse.org/bugs/show_bug.cgi?id=545033). This enables some usage of AspectJ 8 operating on jars containing some of these features from later Java (but where the main set of classfiles in a jar are Java8 or lower). For proper treatment of Java9 code, please use AspectJ 1.9 or later.

1.8.14 available 6-Mar-2019
