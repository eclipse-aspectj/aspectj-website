AspectJ 5 introduces several important changes to the language and tools in order to support Java 5 and extend the capabilities of AspectJ in general. For language changes, see AspectJ 5 Developer’s Notebook. For tools changes, see the AspectJ Development Environment Guide, especially the section on Load-time weaving. The changes are summarized in the FAQ entry on AspectJ 5.

Some of the other documentation has not been updated for AspectJ 5. For the latest documentation, see the AspectJ documentation home page.

For information on bug fixes in AspectJ 5 v1.5.0, see the changes document.
